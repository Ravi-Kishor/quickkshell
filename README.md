# quickshell-git (Prebuilt Arch Package)

Prebuilt `quickshell-git` package built automatically via GitHub Actions.


---

## Install / Update

```bash
curl -LO https://raw.githubusercontent.com/Ravi-Kishor/quickkshell-git/main/update-quickshell
chmod +x update-quickshell
sudo mv update-quickshell /usr/local/bin/update-quickshell
update-quickshell
